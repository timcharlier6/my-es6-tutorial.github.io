Arrow Functions

Arrow function allows to write shorter function syntax.

Code Output 
const hello = val => "Hello " + val;
console.log(hello("world"));

Note:
The handling of this is also different in arrow functions compared to regular functions.
In regular functions the this keyword represents the object that calls the function, which could be the window, the document, a button or whatever.
With arrow functions the this keyword always represents the object that defined the arrow function.

Class

A class is a type of function declared with class that can have properties assigned inside a constructor method.

Code Output 
class Car {
  constructor(name) {
    this.brand = name;
  }

  present() {
    return 'I have a ' + this.brand;
  }
}

class Model extends Car {
  constructor(name, mod) {
    super(name);
    this.model = mod;
  }

  show() {
    return this.present() + ', it is a ' + this.model;
  }
}

const mycar = new Car("Ford");
const mymodel = new Model("Ford", "Mustang");

console.log(mycar.present());
console.log(mymodel.show());
