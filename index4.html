ES6 BASICS

Spread operator allows to quickly copy all or part of an existing array or object into another array or object.

Code Output 

const numbers1 = [1, 2, 3];
const numbers2 = [4, 5, 6];
const numbers1and2 = [...numbers1, ...numbers2];

console.log(numbers1and2);

Destructuring + spread operator

Code Output 
const [one, two, ...rest] = numbers1and2;

console.log(one);
console.log(two);
console.log(rest);
